二叉搜索树最近公共祖先

原理与简单优化

题目

原题链接

https://leetcode-cn.com/problems/er-cha-sou-suo-shu-de-zui-jin-gong-gong-zu-xian-lcof/

分析：

二叉搜索树，由其“搜索某节点，必定先经过其祖先”的特性，可在搜索节点过程中记录下其祖先。如搜索节点4的过程中，即可依次记录下其祖先节点：6 –> 2
由于题目需要求解的是最近公共祖先，因此只需对两个节点的祖先进行比对，而从节点本身开始比对，再依次按父节点、祖父节点等等的顺序进行比对，最先匹配的节点便是最近公共祖先。
此处，可以使用两个STL的stack放置祖先，因为节点本身也定义为它的祖先，所以节点本身也需要入栈。比对的过程从栈顶开始。
但是因为此处使用两个循环进行比对，因此选用一个stack，一个vector。每次从stack中弹出栈顶，再与vector中的元素一一比对，注意vector中元素并不进行删除！

笔者最初解题时，搜索节点的主体代码编写如下，性能一般。

if(t->val == p->val)  /* 1 */
    break;
iSt.push(t);
if(p->val < t->val)   /* 2 */
  t = t->left;
else
  t = t->right;

考虑到代码重复读取节点的val，每次循环中，第一次读取节点的val后，val可能被放到寄存器和cache中，也可能还在主存中，这可能会限制代码的性能。而且，从汇编层面看，1处判断t->val与p->val是否相等的if语句、2处判断二者哪个大的if语句及其分支，这一过程是冗余不必要，因为在汇编中会重复做t->val 与p->val的减法运算。因此，可以对代码做以下的优化。

tmp = p->val - t->val;
if(tmp==0) 
    break;
else if(tmp<0){
    iSt.push(t);
    t = t->left;
}else{
    iSt.push(t);
    t = t->right;
}

以上，分别读取p节点和q节点的val，然后根据减法结果进行分支，避免了不必要的多次读取节点val和汇编层面的反复减法运算。

优化理论依据：

  • 1.这类优化包括识别要执行多次（例如在循环里）但是计算结果不会改变的计算。 —《深入理解计算机系统》P350
  • 2.消除不必要的内存引用。 —《深入理解计算机系统》P354

具体代码

class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {

        stack<TreeNode*> iSt;
        vector<TreeNode*> iVec;
        TreeNode* t;
        t = root;

        int tmp = 0;
        /* find p and trace p's ancestors */
        while(t!=NULL && t->val != p->val){
            /*This part was optimized*/
            tmp = p->val - t->val;
            if(tmp==0)
                break;
            else if(tmp<0){
                iSt.push(t);
                t = t->left;
            }else{
                iSt.push(t);
                t = t->right;
            }
        }
            if(t->val == p->val)
                iSt.push(t);
            else
                return NULL;

        /* find q and trace q's ancestors */
            t = root;
        while(t!=NULL && t->val != q->val){
            tmp = q->val - t->val;
            if(tmp==0)
                break;
            else if(tmp<0){
                iVec.push_back(t);
                t = t->left;
            }else{
                iVec.push_back(t);
                t = t->right;
            }
        }   
            if(t->val == q->val)
                iVec.push_back(t);
            else
                return NULL;

        TreeNode* e;
        tmp = 0;
        int i;
        /* Compare p's ancestors and q's one by one */
        while(!iSt.empty()){
            t = iSt.top();
            i = iVec.size();
            while(i>0){
                e = iVec[i-1];
                if(t->val == e->val)
                    return e;
                i--;
            }
            iSt.pop();
        }


         if(e->val == t->val)
             return e;
         else
             return NULL;

    }
};
