RISC-V汇编详解之？？

这篇文章主要介绍RISC-V汇编的？？，主要参考来源为MIT6.004课程[1]，以及自己的一些笔记。

简单表达式的编译过程

  • 首先，将变量存入寄存器中
  • 将所作的运算编译成对应的计算指令
  • 对于小常数，使用R-I指令处理
  • 对于大常数，使用li伪指令处理
  • C例程：
    int x,y,z;
    y = (x+3) | (y+123456);
    z = (x*4)^y;
  • 对应的RISC-V汇编代码
    (x:10 y:x11 z:x12
    x13,x14用于存储临时值)
    addi x13,x10,3
    li x14,123456
    add x14,x11,x14
    or x11,x13,x14
    slli x13,x10,2
    xor x12,x13,x11
  • 注解：I版本RISC-V没有乘法指令，可以通过左移位实现。如slli x13,x10,2，将x左移2位，从而实现x*4。

条件表达式

1.if表达式
将if表达式编译成分支结构

  • C例程
    int x,y;
    …
    if(x<y){

    y = y - x;

    }

  • RISC-V汇编
    // x:x10 y:x11
    slt x12, x10, x11
    beqz x12, endif
    sub x11, x11, x10
    endif:

  • RISC-V汇编（我们还可以将比较和分支结合起来）
    bge x10, x11, endif
    sub x11, x11, x10
    endif:

  • 注解：“slt x12, x10, x11 ” 当x（x10）< y(x11)时，x12被置为1；“beqz x12, endif” 当x<y不成立时，亦即x>=y,跳转到endif执行。“bge x10, x11, endif ” 当x(x10) >= y(x11)时，跳转到endif执行。

RISC-V汇编详解之基本指令

前言：之前读CSAPP学了X86汇编，最近又学了RISC-V汇编。X86汇编、RISC-V汇编恰好分别是CISC和RISC的典型代表。因此打算写几篇博客介绍一下RISC-V汇编和X86汇编，然后将二者的特性做一个横向对比。

这篇文章主要介绍RISC-V汇编的基本指令，主要参考来源为MIT6.004课程1，以及自己的一些笔记。

RISC-V ISA

1.ISA可以理解为是一种软件和硬件之间的约定。

  • 规定了运算和存储的位置
  • 是对软件如何访问硬件的精确描述

2.RISC-V ISA概述

  • 一种全新的、开源免费的ISA，源于加州伯克利
  • 有几种变种：
    • RV32,RV64,RV128三种不同字长
    • I：整数指令
    • M：乘法和除法指令
    • F和D：单精度、双精度浮点数指令
    • 以及许多其他模块化扩展
